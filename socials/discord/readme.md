Discord — Folder Guide (/socials/discord/readme.md)

This folder contains all content prepared for Discord, including:

What Goes Here

Server announcements

Community updates

Event reminders

Patch notes

Giveaways

FAQ messages

Bot alerts

Welcome messages


Posting Format

Add each new message as a new daily entry:

## Update – 2025-02-10
Your Discord announcement here…

## Update – 2025-02-11
Another update…

Everything stays inside a single file — no numbering needed.

Purpose

This folder is for all Discord communications ready to be copied into the server channels.
