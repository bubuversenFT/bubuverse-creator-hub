Telegram — Folder Guide (/socials/telegram/readme.md)

This folder contains all content created for Telegram channels or groups.

What Goes Here

Daily short updates

Announcements

Quick text posts

Short explanations

Links to articles

New feature alerts

Image captions

Calls-to-action


Posting Format

Each day, simply add a new section:

## Day 01 – 2025-02-10
Your Telegram message here…

## Day 02 – 2025-02-11
Your next message…

No need for separate files.
Just keep adding chronological entries.

Purpose

A single place to store all Telegram-ready content so you can quickly publish it later.
